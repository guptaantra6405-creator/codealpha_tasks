# Unemployment Analysis with Python

This project is completed as part of the CodeAlpha Data Science Internship.  
It focuses on analyzing unemployment trends in India, with special emphasis on the impact of the Covid-19 pandemic.

## Objective
The objective of this project is to analyze unemployment rate data representing the percentage of unemployed people, identify trends and seasonal patterns, and study the impact of Covid-19 on unemployment rates.

## Dataset
The dataset is sourced from Kaggle and contains monthly unemployment statistics across different regions of India.

## Key Analysis Performed
- Data cleaning and preprocessing
- Unemployment trend analysis
- Covid-19 impact analysis
- Regional unemployment comparison
- Seasonal trend identification

## Key Findings
- Unemployment increased sharply during the Covid-19 period.
- Post-Covid recovery is visible but remains unstable.
- Certain regions consistently show higher unemployment rates.
- Seasonal fluctuations indicate dependence on temporary employment.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
